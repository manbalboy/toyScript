# script 영역 안에 하나씩 채워 넣고 설명은 필수 입니다. ## 정규표현식 영역 ### 패턴과 플래그 - 문법 regexp = new RegExp("pattern", "flags"); regexp = /pattern/; // 플래그가 없음 regexp = /pattern/gmi; // 플래그 g, m, i가 있음 - match() : 검색 - replace() : 치환 -기호 교체 문자열의 작업 $& 전체 일치를 삽입합니다. $` 일치 전에 문자열의 일부를 삽입합니다. $' 일치 뒤에 문자열의 일부를 삽입합니다. $n n1-2 자리 숫자 인 경우 n 번째 캡처 그룹의 내용을 삽입합니다. 자세한 내용은 그룹 캡처 를 참조하십시오. $ 주어진 괄호 안의 내용을 삽입합니다. name자세한 내용은 그룹 캡처 를 참조하십시오. $$ 문자 삽입 $ - regexp.test(str) : 일치 여부 확인하기 - 정규식 문자 클래스 \d – 숫자 \D – 숫자가 아닌 문자 \s – 스페이스, 탭, 줄 바꿈 문자 \S – \s를 제외한 모든 문자 \w – 라틴 문자, 숫자, 밑줄 '_' \W – \w를 제외한 모든 문자 . – 정규 표현식 's' 플래그가 있으면 모든 문자, 없으면 줄 바꿈 \n을 제외한 모든 문자
